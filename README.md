# 第一天的学习内容

## 学习目标

- 了解 Claude Code 的基本用法
- 安装和配置 Git
- 关联 GitHub 账号
- 创建第一个仓库并推送代码

## 完成事项

1. 安装了 Git v2.54.0
2. 配置了 Git 用户信息（用户名、邮箱）
3. 使用 HTTPS + Personal Access Token 认证方式关联了 GitHub 账号
4. 创建了本仓库 `first-day-learning`

## 学到的基础命令

| 命令 | 说明 |
|------|------|
| `git init` | 初始化本地仓库 |
| `git clone <url>` | 克隆远程仓库 |
| `git add .` | 添加所有文件到暂存区 |
| `git commit -m "消息"` | 提交变更 |
| `git push` | 推送到远程仓库 |
| `git config --global` | 配置全局设置 |
