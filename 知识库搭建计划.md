# 知识库搭建计划

## 目标
在 `first-day-learning` 仓库中搭建 Markdown 知识库，用 Git 管理学习笔记。

## 任务

- [ ] 任务 1: 创建目录结构（Git/、Claude-Code/、编程语言/）→ 验证: `ls` 能看到三个子目录
- [ ] 任务 2: 在 Git/ 下写第一篇笔记 `Git-基础命令.md` → 验证: 文件内容包含已学的 git 命令总结
- [ ] 任务 3: 在 Claude-Code/ 下写第一篇笔记 `内置skill介绍.md` → 验证: 文件列出已安装的 skill 及作用
- [ ] 任务 4: 更新 README.md 添加知识库导航 → 验证: README 有新目录链接
- [ ] 任务 5: 提交并推送到 GitHub → 验证: `git push` 成功，GitHub 上能看到新文件

## 完成标准
- [ ] 仓库有清晰的分类目录
- [ ] 至少 2 篇笔记已写好
- [ ] 已推送到 GitHub
